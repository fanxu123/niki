
##Niki ?

`Niki` 是一个基于`markdown`的`wiki`管理程序。

##功能

* 基于`文件`，简单易搭建。
* `所见即所得`的markdown编辑器
* 动态生成`文档结构`，易于浏览。
* `语法高亮`
